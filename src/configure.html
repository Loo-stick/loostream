Mode de proxy

☁️ MediaFlow Proxy Recommandé

Les flux passent par un serveur MediaFlow externe. Économise la bande passante du serveur addon. Idéal pour un usage partagé ou public.

📦 Installer MediaFlow Proxy

🏠 Proxy Local Avancé

Les flux passent directement par ce serveur. Aucune dépendance externe. Réservé à un usage personnel (1-3 utilisateurs max).

⚠️ Attention : En mode local, TOUT le trafic vidéo passe par ce serveur. Un flux 4K consomme ~25 Mbps. Assurez-vous d'avoir une bande passante suffisante. Non recommandé pour un usage public.
